# gabrielgingras.me

Personal portfolio and blog built with Vue 3 + Vite.

## Setup

```bash
npm install
npm run dev
```

## Stack

- Vue 3
- Vue Router
- Vite
- Tailwind CSS

## Blog

Blog posts are written in Markdown with frontmatter, located in `src/content/blog/`. Bilingual (FR/EN) content is split within each file using a `<!-- en -->` separator.
